# Purpose

This document defines the Dataset Versioning subsystem of the Zanistarast Dataset Manager.

The Dataset Versioning subsystem provides deterministic, certified, and reproducible management of dataset evolution while preserving logical consistency, dependency integrity, certification continuity, and compatibility with the Certified Core and Rasterast Mathematics.

Every versioning operation shall be deterministic, reproducible, explainable, certifiable, and fully auditable.
