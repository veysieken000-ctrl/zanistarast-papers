Version: 1.0

Status: Architecture

# Purpose

This document defines the reference architecture of the Zanistarast Dataset Manager.

The Dataset Manager provides deterministic, certified, and reproducible management of scientific datasets, metadata, versions, validation, certification, and lifecycle operations while preserving compatibility with the Certified Core and Rasterast Mathematics.

Every dataset operation shall be deterministic, reproducible, explainable, certifiable, and fully auditable.
