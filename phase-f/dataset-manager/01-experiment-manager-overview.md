Version: 1.0

Status: Architecture

# Purpose

This document defines the reference architecture of the Zanistarast Experiment Manager.

The Experiment Manager provides deterministic, certified, and reproducible management of scientific experiments, execution workflows, monitoring, validation, certification, and lifecycle operations while preserving compatibility with the Certified Core and Rasterast Mathematics.

Every experiment operation shall be deterministic, reproducible, explainable, certifiable, and fully auditable.
