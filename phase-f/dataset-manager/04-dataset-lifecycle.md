# Purpose

This document defines the Dataset Lifecycle of the Zanistarast Dataset Manager.

The Dataset Lifecycle provides deterministic, certified, and reproducible management of datasets throughout their entire existence while preserving logical consistency, dependency integrity, certification continuity, and compatibility with the Certified Core and Rasterast Mathematics.

Every lifecycle operation shall be deterministic, reproducible, explainable, certifiable, and fully auditable.

# Dataset States

Every dataset exists in one of the following states

• Draft

• Submitted

• Under Verification

• Certified

• Published

• Active

• Revised

• Archived

• Preserved

State transitions remain deterministic.
