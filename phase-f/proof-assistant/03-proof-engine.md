# Purpose

This document defines the Proof Engine of the Zanistarast Proof Assistant.

The Proof Engine provides deterministic, certified, and reproducible execution of formal proofs while preserving dependency integrity, logical consistency, certification continuity, and compatibility with the Certified Core and Rasterast Mathematics.

Every proof execution shall be deterministic, reproducible, explainable, certifiable, and fully auditable.
