# Purpose

This document defines the formal Proof Language of the Zanistarast Proof Assistant.

The Proof Language provides deterministic, certified, and reproducible representation of axioms, definitions, lemmas, theorems, proofs, and mathematical reasoning while preserving compatibility with the Certified Core and Rasterast Mathematics.

Every proof expression shall be deterministic, reproducible, explainable, certifiable, and fully auditable.
