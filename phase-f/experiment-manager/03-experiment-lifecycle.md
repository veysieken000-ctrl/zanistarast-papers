# Purpose

This document defines the Experiment Lifecycle of the Zanistarast Experiment Manager.

The Experiment Lifecycle provides deterministic, certified, and reproducible management of scientific experiments throughout their complete existence while preserving logical consistency, dependency integrity, certification continuity, and compatibility with the Certified Core and Rasterast Mathematics.

Every lifecycle operation shall be deterministic, reproducible, explainable, certifiable, and fully auditable.

# Experiment States

Every experiment exists in one of the following states

• Draft

• Proposed

• Under Verification

• Certified

• Scheduled

• Running

• Completed

• Published

• Archived

• Preserved

State transitions remain deterministic.
