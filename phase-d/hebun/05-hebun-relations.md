# Integrity Relation

Notation

≡I

Definition

Two Hebûn Elements satisfy the Integrity Relation if both preserve

• certified identity,

• dependency continuity,

• integrity state,

• certification.

---

# Preservation Relation

Notation

≼P

Definition

h₁ ≼P h₂

means

h₂ preserves every certified property inherited from h₁.

No certified property may be lost.

---

# Stability Relation

Notation

≈S

Definition

Two Hebûn Elements are stability-equivalent if both remain mathematically stable under identical certified transformations.

---

# Dependency Relation

Notation

≺

Definition

h₁ ≺ h₂

means

h₂ depends upon h₁.

Properties

• asymmetric

• transitive

• acyclic

---

# Certification Relation

Notation

⊨

Definition

Core ⊨ h

means

h satisfies every certification rule established by the Certified Core.

# Persistence Relation

Notation

⇝

Definition

h₁ ⇝ h₂

means

h₂ is a certified continuation of h₁ through deterministic transformation.

Persistence requires

• identity continuity,

• dependency continuity,

• integrity continuity.

# Relation Properties

Every certified Hebûn Relation shall preserve

• deterministic interpretation,

• integrity,

• persistence,

• dependency,

• Rasterast compatibility.

# Forbidden Relations

• invalidate certified integrity,

• introduce dependency cycles,

• create uncertified persistence,

• bypass certification,

• destroy deterministic traceability.
