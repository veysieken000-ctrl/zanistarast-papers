Notation

• organizational completeness,

• architectural integrity,

• dependency preservation,

• certification.

# Dependency Metric

Notation

dD(m)

Definition

# Stability Metric

Notation

dT(m)

Definition

Measures the resistance of a certified structure to certified structural evolution.

Stable structures preserve

• architecture,

• organization,

• dependency,

• certification.

# Certification Metric

Notation

dC(m)

Definition

Evaluates compatibility with the Certified Core.

Possible outputs

• Certified

• Under Verification

• Not Certified

# Metric Properties

Every certified metric shall satisfy

• determinism,

• reproducibility,

• organizational consistency,

• dependency preservation,

• Rasterast compatibility.

# Forbidden Metrics

Metrics shall never

• ignore structural inconsistency,

• invalidate dependency,

• bypass certification,

• generate non-reproducible evaluations,

• replace deterministic verification.
