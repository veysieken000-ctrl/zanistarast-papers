# Purpose

This document defines the Tutorials Framework of the Zanistarast Community & Governance framework.

The Tutorials Framework provides deterministic, certified, and reproducible educational guidance for learning, implementing, validating, and extending the Zanistarast ecosystem while preserving compatibility with the Certified Core and Rasterast Mathematics.

Every tutorial shall be deterministic, reproducible, explainable, certifiable, and fully auditable.
