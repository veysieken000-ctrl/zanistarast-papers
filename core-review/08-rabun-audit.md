# Working Definition

Rabûn denotes certified governance.

Governance is considered Rabûn only when every decision,

coordination,

rule,

and execution is derived from certified preserved knowledge (Mabûn) while preserving the deterministic dependency chain.

Governance without certified preservation cannot be considered Rabûn.
