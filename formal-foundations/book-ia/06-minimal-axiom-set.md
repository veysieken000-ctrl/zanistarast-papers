# Chapter 06 — Minimal Axiom Set

Version: 1.0

---

# Purpose

This chapter investigates whether the mathematical foundation of Book I can be generated from a smaller set of core axioms.

The objective is to identify the minimal sufficient axiomatic basis while preserving the intended scope of the framework.

---

# Definition

A minimal axiom set is a collection of axioms satisfying the following properties:

1. Every theorem of Book I remains derivable.
2. No axiom is redundant.
3. Removing any remaining axiom changes the derivable results.

---

# Evaluation Procedure

For each core axiom:

- Temporarily remove the axiom.
- Re-evaluate all lemmas.
- Re-evaluate all theorems.
- Record any loss of derivability.

---

# Acceptance Criteria

An axiom belongs to the minimal basis if:

- it is necessary,
- it is independent,
- it contributes to at least one essential theorem.

---

# Expected Outcome

The result of this chapter is not a new mathematical theory.

It is a simplified and more rigorous presentation of the existing framework.

---

# Scientific Review Note

Reducing the number of primitive assumptions improves clarity, facilitates formal verification and strengthens the mathematical presentation.
