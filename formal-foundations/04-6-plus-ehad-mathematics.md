# Purpose

This chapter introduces the 6+Ehad model of Zanistarast.

Unlike the 6+Tek model,

which describes the ontological architecture of a created human being,

the 6+Ehad model describes the ontological dependency of the entire created order upon Ehad.

The model does not place Ehad inside creation.

Rather,

it establishes that every created ontological layer ultimately depends upon Ehad.

---

# Fundamental Principle

Every created universe possesses ontological layers.

However,

none of these layers possesses independent existence.

Their existence depends absolutely upon Ehad.

Therefore,

the cosmological model is expressed as

6 + Ehad.

# Meaning of Ehad

Ehad is not a seventh ontological layer.

Ehad is not one member of the hierarchy.

Ehad is the absolute source upon which the entire hierarchy depends.

Therefore,

Ehad never becomes an element of the created universe.

Ehad remains ontologically unique and incomparable.

---

# Ontological Dependency

Every created layer depends upon the continued existence of Ehad.

This dependency is absolute.

If the dependence upon Ehad were removed,

no created ontological layer could remain.

Therefore,

the existence of the created order is contingent,

whereas Ehad is necessary.
