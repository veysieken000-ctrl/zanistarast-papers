# Purpose

This chapter introduces propositions that follow directly from the primitive definitions and the core axioms.

Unlike theorems, these propositions require only straightforward reasoning and serve as intermediate results for later lemmas and theorems.

---

# Proposition 10.1

Every Rasterast-verified object has previously satisfied Rabûn.

Justification:

Direct consequence of Core Axiom 5.

---

# Proposition 10.2

Every Rabûn object originates from Mabûn.

Justification:

Direct consequence of Core Axiom 4.

---

# Proposition 10.3

Every Mabûn object originates from Zanabûn.

Justification:

Direct consequence of Core Axiom 3.

---

# Proposition 10.4

Every Zanabûn object originates from Hebûn.

Justification:

Direct consequence of Core Axiom 2.

---

# Proposition 10.5

Every Rasterast-verified object preserves its originating Yek.

Justification:

Each layer preserves Yek by definition.

# Proposition 10.6

The deterministic chain preserves identity.

Justification:

No layer modifies the originating Yek.
