# Purpose

This document defines the Stable Public Release architecture of the Zanistarast ecosystem.

The Stable Public Release provides deterministic, certified, and reproducible public distribution of the Zanistarast platform while preserving compatibility with the Certified Core and Rasterast Mathematics.

Every public release shall remain deterministic, reproducible, explainable, certifiable, and fully auditable.
