# Purpose

This document defines the Software Development Kit (SDK) of the Zanistarast ecosystem.

The SDK provides deterministic, certified, and reproducible interfaces for building applications, extensions, scientific tools, and AI-native systems while preserving compatibility with the Certified Core and Rasterast Mathematics.

Every SDK operation shall remain deterministic, reproducible, explainable, certifiable, and fully auditable.
