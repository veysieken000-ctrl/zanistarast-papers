# Purpose

This document defines the Command Line Interface (CLI) of the Zanistarast ecosystem.

The CLI provides deterministic, certified, and reproducible command-line access to runtime services, verification engines, scientific tools, package management, and deployment workflows while preserving compatibility with the Certified Core and Rasterast Mathematics.

Every command execution shall remain deterministic, reproducible, explainable, certifiable, and fully auditable.
