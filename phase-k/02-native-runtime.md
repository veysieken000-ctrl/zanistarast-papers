# Purpose

This document defines the Native Runtime architecture of the Zanistarast ecosystem.

The Native Runtime provides deterministic execution of all certified Zanistarast components while preserving compatibility with the Certified Core and Rasterast Mathematics.

Every execution shall remain deterministic, reproducible, explainable, certifiable, and fully auditable.
