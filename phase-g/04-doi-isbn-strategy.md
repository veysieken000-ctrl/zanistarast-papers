# Purpose

This document defines the DOI and ISBN Strategy of the Zanistarast Academic Ecosystem.

The DOI and ISBN Strategy provides deterministic, certified, and reproducible identification of publications, books, datasets, software releases, and scientific artifacts while preserving compatibility with the Certified Core and Rasterast Mathematics.

Every publication identifier shall be deterministic, reproducible, explainable, certifiable, and fully auditable.

DOIs support

• journal publications,

• conference papers,

• technical reports,

• datasets,

• software releases,

• supplementary materials.

# ISBN Strategy

• books,

• edited volumes,

• monographs,

• textbooks,

• reference manuals,

• collected works.

Each ISBN permanently identifies one certified edition.
