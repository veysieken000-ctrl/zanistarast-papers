# Purpose

This document defines the reference architecture of the Zanistarast Knowledge Graph.

The Knowledge Graph provides the deterministic semantic foundation for representing, organizing, verifying, certifying, and evolving knowledge across the entire Zanistarast ecosystem.

Unlike conventional knowledge graphs, every entity, relationship, and inference shall remain deterministic, reproducible, explainable, auditable, and compatible with the Certified Core.
