# Purpose

This document defines the reference architecture of the Zanistarast Native Agent.

The Native Agent is a deterministic autonomous software entity capable of perceiving information, reasoning over certified knowledge, planning deterministic actions, executing verified operations, and learning from certified experience while remaining fully compatible with the Certified Core.

Unlike conventional AI agents, every decision is deterministic, reproducible, auditable, and verified through Rasterast Mathematics.
