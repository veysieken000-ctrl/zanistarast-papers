# Purpose

This document defines the reference architecture of the Zanistarast Integrated Development Environment (IDE).

The IDE provides a deterministic, certified, and reproducible development environment for building, validating, certifying, and maintaining Zanistarast applications while preserving compatibility with the Certified Core and Rasterast Mathematics.

The IDE unifies editing, compilation, validation, debugging, certification, and deployment into a single deterministic workflow.
