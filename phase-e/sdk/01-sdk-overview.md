# Purpose

This document defines the reference architecture of the Zanistarast Software Development Kit (SDK).

The SDK provides deterministic, certified, and reproducible interfaces for developing applications on top of the Zanistarast ecosystem while preserving compatibility with the Certified Core and Rasterast Mathematics.

The SDK enables developers to build trustworthy software by integrating Native Runtime, Native Compiler, Native Agent, Knowledge Graph, and Rasterast Validator through standardized APIs.
