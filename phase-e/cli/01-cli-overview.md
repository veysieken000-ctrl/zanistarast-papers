# Purpose

This document defines the reference architecture of the Zanistarast Command Line Interface (CLI).

The CLI provides deterministic, certified, and reproducible command-line access to every major component of the Zanistarast ecosystem while preserving compatibility with the Certified Core and Rasterast Mathematics.

The CLI enables developers, researchers, administrators, and automated systems to build, validate, certify, inspect, and manage Zanistarast projects through standardized commands.
