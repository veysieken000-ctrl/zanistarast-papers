# Purpose

This document defines the reference architecture of the Zanistarast Rasterast Validator.

The Rasterast Validator is the deterministic verification core of the Zanistarast ecosystem. It validates every certified object, mathematical definition, ontology, execution plan, agent decision, and knowledge transformation while preserving compatibility with the Certified Core.

Unlike conventional validation systems, every validation result shall be deterministic, reproducible, explainable, certifiable, and fully auditable.
