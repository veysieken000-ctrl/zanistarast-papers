# Purpose

This document defines the Distribution Packages of the Zanistarast Stable Release.

The Distribution Packages provide deterministic, certified, and reproducible packaging of all official Zanistarast components while preserving compatibility with the Certified Core and Rasterast Mathematics.

Every package shall be deterministic, reproducible, explainable, certifiable, and fully auditable.
